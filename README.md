# 🖼️ Image Gallery Project

A responsive and visually appealing Image Gallery website built using HTML, CSS, and JavaScript.  
Users can browse images in a clean grid layout with interactive effects.

## 🚀 Features
- Responsive design for different screen sizes
- Image grid layout
- Hover effects and smooth transitions
- Modern UI design
- Fast and lightweight

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
ImageGallery/
│── index.html
│── style.css
│── script.js
│── images/
│── README.md

## ▶️ How to Run
1. Clone the repository
```bash
git clone <repo-link>
```

2. Open project folder

```bash
cd ImageGallery
```

3. Open `index.html`
or use Live Server in VS Code.
